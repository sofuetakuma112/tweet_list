# Tweet List

## Build Setup

``` bash
# backendディレクトリに移動
cd backend

# package.jsonからdependenciesをインストール
npm install

# APIサーバー起動
npm start

# 1レベル上の階層に移動
cd ..

# frontendディレクトリに移動
cd frontend

# package.jsonからdependenciesをインストール
npm install

# localhost:8080で開発サーバーを立ち上げ
npm run dev
```
